## General info

This is Front-End application created with ReactJs that searches for existing businesses by the given location
via Yelp API.

## Technologies

Project is created with:

- ReactJs
- Redux
- NextJs
- Tailwind CSS

## Other

Be aware that Yelp API doesn't support CORS policy, therefore you're encoureged to visit https://cors-anywhere.herokuapp.com/corsdemo in order to access the full funcitonality of the application.

Yelp APIs used:

- /businesses/search
- /autocomplete
